## NECESSIDADE DE AJUSTES
1- Na parte de cautelas do armamento precisa conter a sua atual situação ( se está cautelado ou nao para evitar a cautela do mesmo armamento)
2- No material é necessário impedir a cautela caso não tenha mais material disponível ou até nao mostra-lo para cautelar
3- No armamento deve ter no banco a finalidade da cautela ( Missão, serviço...)






## FUNCIONALIDADES PARA IMPLEMENTAÇÃO
* FURRIEL
- Controle do combustível
- Controle de munição

* MANUTENÇÕES DO ARMAMENTO
- Dia da manutenção
- Tipo da manutenção
- Id do armamento

* PESSOAL
- Armazenar no banco latitude e longitude da residencia dos militares
- Atraves da aplicação apresentar todos no mapa
- Fornecer a distancia de cada militar para o batalhão
